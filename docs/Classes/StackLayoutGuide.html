StackLayoutGuide

open class StackLayoutGuide<Parent> : LayoutGuide<Parent>, AdjustableLayoutElement, AdaptiveLayoutElement where Parent : LayoutElementsContainer

StackLayoutGuide layout guide for arranging items in ordered list. It’s analogue UIStackView. For configure layout parameters use property scheme. Attention: before addition items to stack, need add stack layout guide to super layout element using func add(layoutGuide:) method.
